## 👏 소개

stayfolio를 모티브로한 숙박 예약 사이트
<br />
<br />
<br />

## 💡 서비스

 <br />
👤 소셜(kakao) 로그인 <br />
🎵 숙소에서 감성을 더해줄 음악 추천 <br />
🏠 카테고리 별 숙소 필터링 및 키워드 검색 <br />
🛒 숙소 예약 및 예약 내역 조회 <br />
